# GLC-RAG Info API demos

A **GLC-RAG API** használatához (Python/PHP minták, éles hívások) **regisztráció szükséges** a **[https://glc-rag.hu/](https://glc-rag.hu/)** oldalon. Külső fejlesztőknek először mindig ott kell fiókot létrehozni (vagy admin hozzáférést kapni), utána a scriptekben megadott hitelesítéssel lehet dolgozni.

Example scripts for the Info API (`/admin/info`, chat `/info` mode):

| Directory | Language |
|-----------|----------|
| [python](./python/) | Python (`info_demo.py`) |
| [php](./php/) | PHP (`info_demo.php`) |

Run each demo from its directory after configuring credentials in the script. See the README inside each folder.
